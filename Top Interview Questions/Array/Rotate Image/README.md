# Rotate Image

找规律换位置，以n
```
[
  [ 5, 1, 9,11, 2],
  [ 2, 4, 8,10, 4],
  [13, 3, 6, 7, 8],
  [15,14,12,16, 9],
  [ 4, 8, 8,10, 2],
]
```
为例。

旋转过后为p
```
[
  [ 4, 15, 13, 2, 5 ],
  [ 8, 14, 3, 4, 1 ],
  [ 8, 12, 6, 8, 9 ],
  [ 10, 16, 7, 10, 11 ],
  [ 2, 9, 8, 4, 2 ]
]
```

可以发现，

```
p[0][0] = n[4][0]
p[4][0] = n[4][4]
p[4][4] = n[0][4]
p[0][4] = n[0][0]

p[0][1] = n[3][0]
p[3][0] = n[4][3]
p[4][3] = n[1][4]
p[1][4] = n[0][1]

...
可以推出：
p[x][y] = n[size - y][x], size = 每行元素个数 - 1
```
